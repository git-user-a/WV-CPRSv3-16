Getting Help

You are currently viewing the online Help file.

To see a list of all available Help topics, click the Help Topics button (Alt + T). In the Help Topics popup window, you can use the Index or Find features to search for different topics about Vitals Lite and how to use it.

A list of Access Keys and Shortcut Keys is available here.

For additional information on the Vitals application, select Help | Vitals Web Page from the main Vitals Lite window.

You can also view or download a copy of the User Manual from the VistA Documentation Library (VDL) at http://www.va.gov/vdl/application.asp?appID=107

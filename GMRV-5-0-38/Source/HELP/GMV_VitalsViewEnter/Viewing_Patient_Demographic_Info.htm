Viewing Patient Demographic Information You can view a Patient Inquiry report for the selected patient in the main Vitals Lite window. This information is read-only and cannot be edited in Vitals Lite.
  1. From the File menu, select Patient Inquiry. A patient inquiry report screen opens, listing all available demographic information for this patient.
  2. Click Print to print the patient inquiry report.
  3. Click Close to close the report window.

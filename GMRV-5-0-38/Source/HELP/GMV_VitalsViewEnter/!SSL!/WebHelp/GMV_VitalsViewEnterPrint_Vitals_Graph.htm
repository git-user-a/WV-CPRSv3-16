Printing the Vitals Graph
In the main Vitals Lite window, you can send the vitals data graph to a printer. The graph will be printed as it appears on the screen at the time of printing. See Viewing Vitals Data to select a different date range or graph type, or see Setting Graph Options to change the appearance of the graph.
  1. From the File menu, select Print Graph. A standard Print dialog box opens.
  2. Select a printer and click OK.

Entering Vitals from the Notes tab
For CPRS users who have access to the Encounter button, the Enter Vitals window can also be opened from the Notes tab. This method will skip the main Vitals Lite window.
If you don’t have access to the Encounter button, see Entering Vitals Data for instructions on accessing Vitals Lite through the CPRS Cover Sheet.
  1. In CPRS, go to the Notes tab and then click the Encounter button. In the Location For Current Activities window, select an existing appointment or admission, or create a new visit, then click OK.
In the Encounter Form window, go to the Vitals tab and then click the Enter Vitals button.
  1. The Enter Vitals window opens, displaying the selected patient’s Name, SSN, and DOB at the top of the window, along with the selected Hospital Location and current Date/Time. To use a different location, date/time, or template for this set of vitals:
Click the Read Monitor button to retrieve vital sign readings from a vital sign monitor (VSM) connected to the workstation.
Click the Date/Time button to select a different date or different time.
Click the Hospital Location button to select a different location.
Click the Exp. View button to display or hide the Templates panel. To select a template, click any template name in the list. If no template has been selected, the default System template will be used.
You can also display or hide the latest vitals on file for this patient by clicking the Latest V. button. The vitals data will be shown at the bottom of the Enter Vitals window.
  1. Type a value for each vital sign or measurement, then press Enter to move to the next field. If you would like to select a different qualifier for a vital sign, click the down-arrow button to display a list of its qualifiers.
To enter a non-numeric value for a particular vital sign or measurement, use one of these options.
  1. Click Save. To enter additional vitals for this patient, repeat steps 4–6 using a different date, time, and/or location.
To discard the vitals you have entered, click Cancel. You will be prompted to confirm the cancellation.
When you are finished entering vitals for this patient, click Save and Exit. The Enter Vitals window closes.

Viewing the Data Grid Report
You can view existing vitals data for the selected patient in a text-only report format. This is the same information that is displayed in the data grid. It is read-only and cannot be edited or printed from this window.
  1. In the main Vitals Lite window, select Data Grid Report from the File menu. A report screen opens, showing a text version of the data in the data grid.
  2. To view more or less information, select a different date range in the main Vitals Lite window. See Viewing Vitals Data for more information.
  3. Click Close to close the report window.

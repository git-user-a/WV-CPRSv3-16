Introduction to Vitals Lite
The Vitals application is designed to collect the vital signs and measurements that are associated with a patient's hospital stay or outpatient clinic visit, and store the data in the patient's electronic medical record. Data can be accessed by several VistA (Veterans Health Information Systems and Technology Architecture) applications that interface with the Vitals/Measurements application, such as CPRS, Health Summary, and Pharmacy.
The Vitals Lite module is available through CPRS and is used to enter vitals data, view allergies, and mark incorrect vitals as entered in error.
To open Vitals Lite from the CPRS v26 cover sheet:
Click any item inside the Vitals section of the CPRS Cover Sheet to open Vitals Lite. The Vitals Lite window provides a limited set of Vitals features to CPRS users, and its functionality is supplied by a DLL (Dynamic Link Library) from the Vitals application.
See Appendix C in the Vitals User Manual for more information about Vitals Lite and its features. You can view or download the User Manual from the VistA Documentation Library (VDL) at http://www.va.gov/vdl/application.asp?appID=107
From the main Vitals Lite window, you can:
Enter vitals data
Mark vitals as entered in error
View vitals data on a graph and grid
View a text version of the vitals data grid
View a text version of the vitals graph
Print the vitals data graph
Set graph options
View patient demographic information
View allergies

Break the Clinical Link If your workstation has been configured for CCOW, a clinical link is automatically active, and will remain active unless you break the clinical link. To temporarily turn off CCOW functionality, select Break the Clinical Link from the File menu. The CCOW icon changes to Broken and the VistA applications are unsynchronized, allowing you to work on two different patients when multiple CCOW- compliant applications are open. The clinical context will be automatically restored at your next login, or you can manually rejoin the clinical context if necessary.

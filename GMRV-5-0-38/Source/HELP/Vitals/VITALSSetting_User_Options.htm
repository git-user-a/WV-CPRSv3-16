Setting User Options The User Options feature is used to define the vitals data grid display and the search delay setting. To set user options:
  1. From the File menu in the main Vitals window, select User Options. The User Preferences window opens.
  2. Use the Display Values area to set display options for Normal and Abnormal Values:
Click the Text button to select a text color for the table. Click the Background button to select a background color for the table. Check the Bold check box to bold the text. Check the Show Qualifiers check box to show qualifier abbreviations with each value. To restore default settings, click the Defaults button. Default settings for Normal Values are: Black Text, White Background, Not Bold, Show Qualifiers. Default settings for Abnormal Values are: Red Text, White Background, Not Bold, Show Qualifiers.
  1. Use the Search Value area to set the Search Delay setting. The Search Delay is the amount of time, in seconds, between entering characters and beginning a Patient or Location lookup.
Select 0.5 for the fastest lookup, or 5.0 for the slowest The default Search Delay setting is 1.0
  1. Click OK to save changes, or Cancel to discard changes.
To customize the appearance of the vitals data graph, see Setting Graph Options.

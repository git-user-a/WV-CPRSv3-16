Setting Graph and Grid Options When a patient is selected in the main Vitals window, existing vitals data values are displayed in a graph and a grid on the right side of the window. See Viewing Vitals Data for information about setting date ranges or graph types. You can customize the graph by setting the Graph Options check boxes, by changing the zoom percentage, and by changing the background color. You can customize the grid display by setting user options. To set graph options:
  1. From the View menu, select Graph Options. Four check boxes open on the left side of the graph:
    • Click the Values check box to display a numeric label for each point on the graph.
    • Click the Time Scale check box to view the graph in actual time instead of evenly distributed intervals. You can click on a data value in the graph to see additional information about the vitals that were entered that day.
    • Click the 3D check box to create a 3D effect on the graph. This option is only available when the Time Scale check box has been checked.
    • Click the Allow Zoom check box to zoom the graph view in or out.
  1. To change the zoom percentage, enter a new value in the Zoom Percent text box. This value defines the "zoom in" and "zoom out" buttons (the two magnifying glass icons). Click the Reset Zoom button to reset the Zoom Percent box to its default value (10).
  2. To change the background color of the graph, choose Select Graph Color from the File menu. Click a color in the palette, then click OK to apply the color setting.

Selecting a Patient You must select a patient before you can enter vitals data, view existing data, or view allergies. The Vitals application allows you to select one patient, or multiple patients within a patient group. To select one or more patients:
  1. On the top Vitals tool bar, click the Pt Select button. The Patient Selector pane opens on the left side of the Vitals window.
  2. Under the Patient Groups List, click Unit, Ward, Team, Clinic, or All button to select a group or location:
Unit displays a list of Nursing units Ward displays a list of MAS Wards Team displays a list of teams defined in the OE/RR List file (#100.21) Clinic displays a list of clinics for a predetermined period of time (e.g., Today, Yesterday, Past week) All displays a list of all available patients - enter a patient’s last name or SSN to search for a specific patient
  1. Click a location. A list of patients for the selected group or location displays below the Group List.
  2. To select one patient, double-click a patient name and click OK to confirm the selection. The patient information opens in the Vitals window: see Entering Vitals Data for One Patient for further instructions.
  3. To select more than one patient, hold down the CTRL button and click multiple names, then click the Enter Vitals for Selected Patients button at the bottom left of the window. The first patient’s information opens in the Vitals window: see Entering Vitals Data for Multiple Patients for further instructions.

About Templates Templates are used to customize the Enter Vitals window. Each template is a set of vitals/measurements that has been grouped together to make data entry simpler and easier. For example, if you typically collect only Temperature, Pulse, and Respiration data, you can create a "TPR Only" template. Your system administrator will use the Vitals Manager module to create three types of templates: System templates are available to all system users Division templates are available to all users within a division (for multi-divisional sites) Location templates are available to all users within a particular ward, team, or clinic If your system administrator has enabled the User Templates option, you can create one or more user templates for your own use. User templates are available only to the individual user who created the template. To select a template: Note: In order to open the Enter Vitals window, a patient must be selected.
  1. If the Templates pane is not displayed in the Enter Vitals window, click the Exp. View button. The Templates pane opens, displaying a list of System, Division, Location, and User template folders.
  2. Expand the folders to display all available templates. Note: The yellow icons indicate default templates.
  3. To select a template, click the template name. The vital signs and measurements associated with the template open in the right side of the Enter Vitals window.
See Entering Vitals for One Patient or Entering Vitals for Multiple Patients for further instructions.

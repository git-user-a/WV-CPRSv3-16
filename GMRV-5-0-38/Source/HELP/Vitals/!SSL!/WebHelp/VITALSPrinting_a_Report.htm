Printing a Report
Four standard reports are available in the Vitals application:
Cumulative Vitals Report Available for single patient, entire ward, or selected rooms on a ward
Latest Vitals by Location Available for an entire ward
Latest Vitals Display for a Patient Available for single patient only
Vitals Entered in Error for a Patient Available for single patient only
These reports cannot be viewed on-screen: they must be printed. They require a device that can handle 80 or more columns.
To print a report:
  1. From the Reports menu in the main Vitals window, select a report. The Report Options window opens, showing the name of the selected report.
  2. In the Include Patient(s) section, select a specific patient, an entire ward, or specific rooms/beds on a ward. (The currently active patient is selected by default.)
  3. In the Date Range section, select a Start Date and an End Date for the selected report (if appropriate). Click the drop-down arrow next to the Start Date and End Date fields to view a calendar. Note: Start and End dates do not apply to the Latest Vitals by Location report or the Latest Vitals Display for a Patient report.
  4. In the Device section, enter a partial device name, then select the appropriate device from the list.
  5. In the Queue To Run At section, select a date and time to print this report (default is today). For obvious reasons, reports cannot be printed for a past date/time.
  6. Click Print to send the report to the selected device.

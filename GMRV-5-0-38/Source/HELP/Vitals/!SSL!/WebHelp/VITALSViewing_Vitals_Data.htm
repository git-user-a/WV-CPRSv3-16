Viewing Vitals Data on the graph and grid
When a patient is selected in the main Vitals window, existing vitals data values are displayed on the right side of the window in a graph (top) and a grid (bottom). The time scale runs from left to right, displaying the most recent values on the far right.
You can select a date range and a type of graph to narrow down the amount of data to be displayed.
If you would like to customize the graph and grid display, see Setting Graph Options and Setting User Options. You can also view the vitals data as a text report.
To select a date range:
Use one of the following methods to select a date range:
  • Click a predetermined time frame from the box on the left of the graph. Today displays todayâ€™s data only, T-1 displays data for today plus one previous day, T-7 displays data for today plus seven previous days, and so on. All Results displays all available data for the selected patient.
  • Click Date Range to select a customized date range, then set "Start with" and "Go to" dates.
All data within the selected date range is displayed in the graph and grid.
To select a type of graph:
Use one of the following methods to select a type of graph:
  • Click a vital type row heading in the grid (for example, Temp or B/P).
  • Select a graph type from the drop-down list at the bottom left corner of the graph.
The selected graph opens above the grid, displaying the data values from the grid.
To print the vitals data graph:
  1. Right-click the graph and select Print Graph. A standard Print dialog box opens.
  2. Select a printer and click OK. The graph is printed as it appears on screen.
See Printing a Report for a list of available reports and printing options.

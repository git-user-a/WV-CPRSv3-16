Deleting a User Template
If the system parameter Allow User Templates in the Vitals Manager module is checked, you may create, edit, or delete your own templates. If Allow User Templates is not checked, user templates are not available.
To delete an existing template:
  1. From the File menu in the main Vitals window, select Edit User Templates. The Vitals Template Editor opens: all existing user templates are listed in the left side of the window.
  2. Click a template name, then click the Delete button at the bottom of the window. A confirmation message appears.
  3. Click OK to confirm the deletion, or click Cancel to keep the template.
For more information on working with user templates, see Creating a User Template and Editing a User Template.

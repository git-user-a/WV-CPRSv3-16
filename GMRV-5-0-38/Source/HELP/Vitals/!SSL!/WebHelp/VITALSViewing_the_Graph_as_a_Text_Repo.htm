Viewing the Graph as a Text Report

You can view data for the selected vitals in a text-only report format. This information is read-only and cannot be edited or printed from this window.

  1. In the main Vitals window, select Graph Report from the Reports menu. A report screen opens, showing a text version of the data in the graph.

  2. To view more or less information, select a different vitals in the main Vitals window.

  3. Click Close to close the report window.

Creating a User Template
If the system parameter Allow User Templates in the Vitals Manager module is checked, you may create or edit your own templates. If Allow User Templates is not checked, you cannot create a user template. (You must have the GMV MANAGER security key to use Vitals Manager: contact your IRMS representative for assistance.)
To create a new template:
  1. From the File menu in the main Vitals window, select Edit User Templates. The Vitals Template Editor opens: any existing user templates are listed in the left side of the window.
  2. Click the New button at the bottom of the window. Enter a descriptive name for the new template (up to 30 characters), then click OK. The new template name appears in the Templates list on the left side of the Vitals Manager window.
  3. Click the new template name. A blank template form opens in the Vitals Template Editor window. Enter a brief description of the template (up to 50 characters).
  4. To add vital types and qualifiers to this template, click the Add button on the far right side of the window. A list of available vital types opens.
  5. Select one or more vital types to be added to this template: use Ctrl+ click to select multiple vital types, or Shift+ click to select a range of vital types. Click OK to add the selected vitals to the Vitals Template Editor window.
  6. In the Template Vitals list, click a vital type to add qualifiers to it. All available qualifiers for the selected vital will appear in the Qualifiers list. If no qualifiers are available for the selected vital type, the Qualifiers list will be empty.
  7. Select a qualifier in each category, or select None by clicking the blank line at the top of the drop down list. Repeat steps 6-7 to add qualifiers to each vital type in the list.
  8. Click Save. The template is added to the Templates list in the Enter Vitals window.
For more information on working with user templates, see Editing a User Template and Deleting a User Template.

Editing a User Template
If the system parameter Allow User Templates in the Vitals Manager module is checked, you may create or edit your own templates. If Allow User Templates is not checked, you cannot create a user template.
To edit an existing user template:
  1. From the File menu In the main Vitals window, select Edit User Templates. The Vitals Template Editor opens: all existing user templates are listed in the left side of the window.
  2. Click a template name. The template opens in the Vitals Template Editor, and any of the following items may be updated:
    • Change the Template Name (up to 30 characters)
    • Change the Template Description (up to 50 characters)
    • To add a new vitals type, click the plus Add button
    • To delete an existing vital type, select the vital type and then click the Delete button
    • To add or modify qualifiers for a vital type, select the vital type and then select or clear the Qualifiers drop down list. If no qualifiers are available for the selected vital type, the Qualifiers list will be empty.
  1. Click the Save button at the bottom of the window. The template is ready to use.
For more information on working with user templates, see Creating a User Template and Deleting a User Template.

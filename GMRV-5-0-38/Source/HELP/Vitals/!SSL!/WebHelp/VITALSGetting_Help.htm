Getting Help
Electronic Help is available in two formats: Online Help and Hints.
You are currently viewing the online Help file. Use the Contents, Index, or Search features to find detailed information about the software and how to use it. In any Vitals window, press F1 for help.
To view Hints, place the cursor on a field, menu item, or button. A brief description of the selected item will display.
A list of Access Keys and Keyboard Shortcuts is available here.
To obtain additional information on the Vitals application, select Help | Vitals Web Site from the main Vitals window. The Vitals User Manual can also be viewed or downloaded from the VistA Documentation Library.(VDL) at http://www.va.gov/vdl/application.asp?appID=107

Viewing the Data Grid as a text report
You can view existing vitals data for the selected patient in a text-only report format. This information is read-only and cannot be edited or printed from this window.
  1. In the main Vitals window, select Data Grid Report from the Reports menu. A report screen opens, showing a text version of the data in the data grid.
  2. To view more or less information, select a different date range in the main Vitals window. See Viewing Vitals Data for more information.
  3. Click Close to close the report window.

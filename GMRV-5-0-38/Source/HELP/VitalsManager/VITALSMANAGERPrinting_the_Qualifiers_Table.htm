Printing the Qualifiers Table

The Qualifiers Table is a list all vitals types and their associated qualifiers and categories. See the Edit Vitals Categories and Qualifiers section for more information.

To print the Qualifiers Table:

4 In the main Vitals Manager window, select Print Qualifiers Table from the File menu. The Device Selection screen appears.

5 Enter a device name in the Device field, then select a date/time in the Queue To Run At field. The default date/time is the current server date/time. This report cannot be queued to run for a past date/time.

6 Click OK to print the report.

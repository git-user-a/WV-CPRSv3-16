Setting System Parameters

Three system parameters are available in the Vitals Manager module: Allow User Templates, Help Menu Web Address, and Version Compatibility.
Important: The VistA server install (KIDS Build) will automatically set the Help Menu Web Address and Version Compatibility parameters for the client/server versions being installed. Do not modify these parameters unless the site is testing a patch or performing local modifications to the client software.
To edit system parameters:
  1. In the main Vitals Manager window, click the System Parameters folder. The three parameters display on the right side of the window.
  2. Click the Allow User Templates check box to allow all clinicians at your site to create and/or edit user templates in the Vitals Manager and Vitals modules, or clear the check box to prevent them from creating their own templates.
  3. Enter a URL for the Vitals Home Page in the Help Menu Web Address to direct the userâ€™s browser to this page.
  4. Click one or more check boxes in the Version Compatibility section to indicate which versions of the Vitals and Vitals Manager client is compatible with the current version of Vitals running on the VistA M server. Because backward compatibility is required, more than one version of each application may be checked.
  5. Click the Save System Parameters button at the top of the window to save changes.

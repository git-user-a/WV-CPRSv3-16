Creating a New Template

This procedure can be used to create any type of template â€“ User, Location, Division, or System.

To create a new template:

  1. From the Templates menu in the Vitals Manager window, select New Template. The "Create New Vitals Template" window opens.

  2. Click a template Type (System, Division, Location, or User), and then enter a name, template name, and brief template description for the new template.

  3. Click OK. The new template name appears in the Templates list on the left side of the Vitals Manager window, and the blank template form opens in the Vitals Template Editor window.

  4. To add vital types and qualifiers to this template, click the Add button on the far right side of the window. A list of available vital types opens.

  5. Select the vital types to be added to this template: use Ctrl+ click to select multiple vital types, or Shift+ click to select a range of vital types. Click OK to add the selected vitals to the Vitals Template Editor window.

  6. In the Template Vitals list, click a vital type to add default qualifiers to it. All available qualifiers for the selected vital will appear in the Qualifiers list. If no qualifiers are available for the selected vital type, the Qualifiers list will be empty.

  7. Select a qualifier in each category, or select None. Repeat steps 6 and 7 to add qualifiers to each vital type in the list.

  8. From the Templates menu, select Save Template. The template is ready to use.

Editing a Template

Use the following procedures to modify or delete any type of template â€“ User, Location, Division, or System. You can also set a default template for each template type: one default System template, one default Division template, etc. (The default System template will be used if no other default templates are set.) To modify an existing template:
  1. In the Vitals Manager window, double-click the Templates folder. The folder expands, showing all available templates.
  2. Click a template name. The template opens in the Vitals Template Editor, and any of the following items may be updated:
    • Change the Template Name (up to 30 characters)
    • Change the Template Description (up to 50 characters)
    • To add a new vitals type, click the Add button
    • To delete an existing vital type, select the vital type and then click the minus Delete button
    • To add or modify qualifiers for a vital type, select the vital type and then select or clear the Qualifiers drop down list. If no qualifiers are available for the selected vital type, the Qualifiers list will be empty.
  1. Select Save Template from the Templates menu. The template is ready to use.
To delete an existing template:
  1. In the Vitals Manager window, double-click the Templates folder. The folder expands, showing all available templates.
  2. Click a template name, then select Delete Template from the Templates menu. A confirmation message appears.
  3. Click OK to confirm the deletion, or click Cancel to keep the template.
To set a default template:
  1. In the Vitals Manager window, double-click the Templates folder. The folder expands, showing all available templates.
  2. Click a template name, then select Set Default Template from the Templates menu. A yellow icon appears next to the template name in the Templates list.
